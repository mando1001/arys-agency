# ARYS Agency - Production Website

Ez az ARYS Agency hivatalos weboldalának forráskódja.

## Tech Stack
- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **UI Components**: Lucide React, Shadcn/ui
- **Backend Integration**: n8n via API Proxy

## Fejlesztés

```bash
npm install
npm run dev
```
